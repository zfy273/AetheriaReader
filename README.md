## 🇨🇳 中文

一款精美、流畅的本地电子书与漫画阅读器。支持多种格式，专为提供沉浸式的阅读体验而设计。

### ✨ 核心功能
- 📖 **多格式支持**：全面支持 TXT 纯文本、PDF 文档以及 ZIP/CBZ 漫画格式。
- 🎨 **精美 UI 设计**：采用 Material Design 3 设计规范，配合流畅的卡片按压回弹动画，体验丝滑。
- 🌗 **护眼主题**：内置“纸质暖白”与“水墨深夜”主题。
- 🌍 **多语言无缝切换**：应用内即时切换语言（中、英、日等），无需重启。
- 📂 **本地书架**：一键导入本地图书，自动记录阅读进度。

### 🚀 如何获取 APK 安装包
1. **通过云端打包**：在 AI Studio 界面右上角的系统莱单中选择 `Generate APKs/AABs` 即可直接下载。
2. **本地环境编译**：若您在 Android Studio 中查看此项目，请在根目录终端运行 `./gradlew assembleDebug`，编译完成后可在 `app/build/outputs/apk/debug/app-debug.apk` 找到应用安装包。

## 🇬🇧 English

A beautiful and smooth local e-book and comic reader. It supports multiple formats and is designed to provide an immersive reading experience.

### ✨ Key Features
- 📖 **Multi-format Support**: Fully supports plain TXT, PDF documents, and ZIP/CBZ comic archives.
- 🎨 **Elegant UI Design**: Adopts Material Design 3 with smooth spring physics animations (like card press bounces) for a buttery-smooth experience.
- 🌗 **Eye-care Themes**: Built-in "Warm Paper" and "Midnight Ink" reading backgrounds.
- 🌍 **Seamless Language Switching**: Switch languages (Chinese, English, Japanese) instantly in-app without restarting.
- 📂 **Local Library**: Easily import local books with automatic reading progress tracking.

### 🚀 How to build/get the APK
1. **Cloud Build**: Click the system menu in the top right corner of the AI Studio interface and select `Generate APKs/AABs` to download directly.
2. **Local Compilation**: If you are building the project in a local development environment, run `./gradlew assembleDebug` in the root terminal. Once compiled, the APK can be found at `app/build/outputs/apk/debug/app-debug.apk`.

## 🇯🇵 日本語

美しく滑らかなローカル電子書籍＆コミックリーダー。複数のフォーマットに対応し、没入感のある読書体験を提供するために設計されました。

### ✨ 主な機能
- 📖 **マルチフォーマット対応**：TXT、PDFドキュメント、およびZIP/CBZコミックアーカイブを完全にサポート。
- 🎨 **美しいUIデザイン**：Material Design 3を採用し、滑らかなカードのバウンスアニメーションなどでシルクのような操作感を実現します。
- 🌗 **目に優しいテーマ**：「ウォームペーパー」や「ミッドナイトインク」の読書用背景を内蔵。
- 🌍 **シームレスな言語切り替え**：アプリを再起動することなく、アプリ内で即座に言語（中国語、英語、日本語）を切り替え可能。
- 📂 **ローカル本棚**：ローカルの書籍を簡単にインポートし、読書進捗を自動的に記録します。

### 🚀 APKの取得方法
1. **クラウドビルド**：AI Studioインターフェースの右上隅にあるシステムメニューから `Generate APKs/AABs` を選択してダウンロードします。
2. **ローカル環境でビルドする場合**：ルートディレクトリのターミナルで `./gradlew assembleDebug` を実行してください。ビルド完了後、APKファイルは `app/build/outputs/apk/debug/app-debug.apk` に生成されます。
